Инструкция к клеточному автомату

Меню управления КА


    Позволяет задавать размерность поля КА, количество машин для случайной генерации (при указании "0" будет сгенерировано пустое поле).
    Выбор правил клеточного автомата возможен из трёх вариантов: двух классических правил оригинальной BML-модели, когда машины расчитываются поочерёдно на каждом шаге (в этом случае приоритет определяется порядком обработки) и двигаются если есть такая возможность, и правила вероятностного разрешения конфликтов, при котором двигаются те автомобили, в чью пользу разрешился конфликт при возможности попадания в одну и ту же клетку синей и красной машин. Изменять правила работы КА можно в процессе его работы, в том числе и указывать новую вероятность приоритетов.
    С помощью кнопки "Шаг" можно выполнить одну итерацию работы КА, с помощью кнопки "Старт/Стоп" можно запустить выполнение итераций с указанием паузы между ними.
    На поле КА кликами по ячейкам можно создавать и удалять сушествующие машины. По клику на пустой ячейке появится синяя машина, по клику на синей - красная, по клику на красной машина будет удалена.
